# Tailwind CSS Basics

## Common Utility Classes

### Spacing
- `p-{n}` - Padding
- `m-{n}` - Margin
- `px-{n}`, `py-{n}` - Horizontal/Vertical padding
- `mx-{n}`, `my-{n}` - Horizontal/Vertical margin

### Flexbox
- `flex` - Display flex
- `items-center` - Align items center
- `justify-between` - Space between items
- `gap-{n}` - Gap between flex items

### Grid
- `grid` - Display grid
- `grid-cols-{n}` - Number of columns
- `gap-{n}` - Gap between grid items

### Typography
- `text-{size}` - Font size (sm, base, lg, xl, 2xl, etc.)
- `font-{weight}` - Font weight (normal, medium, semibold, bold)
- `text-{color}` - Text color

### Colors
- `bg-{color}-{shade}` - Background color
- `text-{color}-{shade}` - Text color
- `border-{color}-{shade}` - Border color

### Borders
- `border` - Add border
- `rounded-{size}` - Border radius
- `shadow-{size}` - Box shadow

## Responsive Design

Breakpoint prefixes:
- `sm:` - 640px and up
- `md:` - 768px and up
- `lg:` - 1024px and up
- `xl:` - 1280px and up
